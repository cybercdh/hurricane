# hurricane

Searches [Hurricane Electric](https://bgp.he.net/) for a given search term and returns all associated IP ranges.

## Installation

Assuming GO is installed...

```bash
go install github.com/cybercdh/hurricane@latest
```

## Usage

```bash
hurricane <search-term>
```
e.g.
```bash
hurricane paypal
```
